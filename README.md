# 🐼 discovering-pandas

## 🚀Goals
- Master DataFrames and Series.
- Practice Data Cleaning (handling missing values, duplicates, and type casting).

## Skills Practiced:

- **Environment Management:** Working with Linux `venv` and `.gitignore`.
