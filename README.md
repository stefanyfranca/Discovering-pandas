## 🎯 Goals
- **Master DataFrames and Series:** Understanding the core structures of Pandas.
- **Data Cleaning:** Practice handling missing values, duplicates, and type casting.
- **EDA (Exploratory Data Analysis):** Perform analysis on real-world datasets such as Global Flights, Airports, and Sales.

## 🛠️ Skills Practiced
- **Data Loading:** Efficiently reading `.csv` and `.json` files.
- **Manipulation:** Advanced filtering, slicing, and sorting.
- **Aggregations:** Statistical summaries with `.describe()` and complex grouping with `.groupby()`.
- **Environment:** Working with Linux `venv`, `.gitignore`, and Jupyter Notebooks.

## 📂 Project Structure
- `scripts/`: Python scripts (`.py`) for automated tasks.
- `notebooks/`: Jupyter Notebooks (`.ipynb`) for interactive exploration and visualization.
- `data/`: Sample datasets (datasets larger than 100MB are ignored by git).
